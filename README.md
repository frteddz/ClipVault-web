# ClipVault

> Clipboard history manager — never lose a copied item again.

This repository contains only the built distribution files. The source code is not publicly available.

## Features

- Automatically records every item you copy
- Search through clipboard history with instant filtering
- Organize clips into custom categories and folders
- Star frequently used items as favorites
- Support for text, images, and rich content
- Pin important clips to keep them from being overwritten
- One-click copy to restore any previous clip
- Clear history or specific categories on demand
- Dark mode and customizable theme

## Why ClipVault?

ClipVault stores everything locally in your browser's IndexedDB — no data is ever sent to the cloud. Your clipboard history stays on your machine, giving you full control over privacy. It's free, lightweight, and designed to be the clipboard manager your browser was missing.
